# Repository Structure

This repository uses a numbered folder structure to organize files by purpose and stage of use.  
The folders were originally named with descriptive prefixes and later simplified to numbers for a cleaner layout.

## Folder Guide

**01 - Archive**  
Previously `01_archive`. Contains legacy or reference files that are no longer actively used but are retained for historical purposes.

**02 - Active**  
Previously `02_active`. Holds files that are currently in use or under active development.

**03 - Styles**  
Previously `03_styles`. Contains styling assets such as CSS, templates, or other design-related resources.

**04 - Sandbox**  
Previously `04_sandbox`. Used for testing, experimentation, and temporary work.
